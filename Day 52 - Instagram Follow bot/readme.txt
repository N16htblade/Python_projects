DAY 52 - Instagram follower bot

Description:
Instagram bot using Selenium that Follows everyone on specified page.

Requirements:
- Python + Selenium
- use class and functions to complete project

Steps:
- Open Instagram.com
- Login and dismiss all notifications and popups;
- Change to page with large number of followers;
- Open list of followers and Follow everyone in order.

Notes:
Save username and passwords as environment variables
